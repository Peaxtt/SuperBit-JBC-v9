# คู่มือการใช้งาน JBC Extension

## เพิ่ม Extension ก่อนเริ่ม

1. เปิด [makecode.microbit.org](https://makecode.microbit.org)
2. สร้าง **New Project**
3. กด **Extensions** → วาง URL นี้ → Enter
   ```
   https://github.com/Peaxtt/Peaxtt-SuperBit-JBC-v1
   ```
4. จะเห็น category **JBC** (สีม่วง) และ **JBCJoystick** (สีชมพู) ในแถบซ้าย

---

## micro:bit มี 2 ตัว

| ตัว | ทำอะไร |
|-----|--------|
| **Robot** | ติดบนหุ่น ควบคุมมอเตอร์และ IMU |
| **Joystick** | ถือในมือ กดปุ่มสั่งหุ่น |

ต้องเขียนโค้ดแยกกันและ flash แยกกันครับ

---

## โค้ดตัวอย่าง Robot

> วาง block ตามลำดับนี้

```
┌─ on start ──────────────────────────────┐
│  JBC init robot group  67               │
│  (รอสักครู่ หุ่นจะแสดง 😴 แล้ว 😊)      │
└─────────────────────────────────────────┘

┌─ when cmd  1  received ─────────────────┐
│  move straight  speed 150  ms 2000      │
└─────────────────────────────────────────┘

┌─ when cmd  2  received ─────────────────┐
│  turn  90  °                            │
└─────────────────────────────────────────┘

┌─ when cmd  3  received ─────────────────┐
│  stop                                   │
└─────────────────────────────────────────┘
```

### อธิบาย block

| Block | ความหมาย |
|-------|----------|
| `JBC init robot group 67` | เตรียมหุ่น — ต้องใส่ไว้ใน **on start** เสมอ เลข 67 คือช่องวิทยุ (ต้องตรงกับ Joystick) |
| `when cmd 1 received` | เมื่อรับคำสั่งหมายเลข 1 จาก Joystick → ทำสิ่งที่อยู่ข้างใน |
| `move straight speed 150 ms 2000` | เดินตรง ความเร็ว 150 เป็นเวลา 2000ms (2 วินาที) |
| `turn 90 °` | หมุนขวา 90 องศา (ใส่ -90 = หมุนซ้าย) |
| `stop` | หยุดมอเตอร์ทันที |
| `IMU ON` | เปิด/ปิด gyro — ถ้าเปิดหุ่นจะเดินตรงอัตโนมัติ |

## โค้ดตัวอย่าง Joystick

```
┌─ on start ──────────────────────────────┐
│  JBC joystick init group  67            │
└─────────────────────────────────────────┘

┌─ on button A pressed ───────────────────┐
│  send cmd  1                            │
└─────────────────────────────────────────┘

┌─ on button B pressed ───────────────────┐
│  send cmd  2                            │
└─────────────────────────────────────────┘

┌─ on button A+B pressed ─────────────────┐
│  send cmd  3                            │
└─────────────────────────────────────────┘
```

### อธิบาย block

| Block | ความหมาย |
|-------|----------|
| `JBC joystick init group 67` | เตรียม Joystick — เลข 67 ต้องตรงกับหุ่น |
| `send cmd 1` | ส่งคำสั่งหมายเลข 1 ไปให้หุ่น |
| `send cmd 2` | ส่งคำสั่งหมายเลข 2 ไปให้หุ่น |
| `send cmd 3` | ส่งคำสั่งหมายเลข 3 ไปให้หุ่น |

---

## วิธีใช้งาน

1. Flash โค้ด **Robot** ลง micro:bit ตัวที่ติดบนหุ่น
2. Flash โค้ด **Joystick** ลง micro:bit ตัวที่จะถือในมือ
3. เปิดไฟหุ่น — รอจนเห็น 😊 (ใช้เวลา ~1 วินาที)
4. กดปุ่ม A บน Joystick → หุ่นเดินตรง 2 วินาที
5. กดปุ่ม B → หมุนขวา 90°
6. กดปุ่ม A+B → หยุด

---

## เพิ่มคำสั่งเองได้

ใส่ตัวเลขอะไรก็ได้ใน `send cmd` และ `when cmd received` — ขอแค่ตัวเลขตรงกัน

```
Joystick กด Logo → send cmd 4

Robot: when cmd 4 received → close gripper
```

## block Tuning (สำหรับครู/ผู้ดูแล)

ไม่จำเป็นต้องแตะถ้าหุ่นทำงานได้ปกติ — ใช้ปรับถ้าหุ่นเดินเบี้ยวหรือหมุนไม่ตรง

| Block | ใช้เมื่อไหร่ |
|-------|-------------|
| `straight PID Kp Ki Kd` | หุ่นเดินเบี้ยวซ้าย/ขวา → ปรับ Kp |
| `turn PID Kp Ki Kd` | หมุนไม่ครบองศา หรือ overshoot → ปรับ Kp |
| `turn rate ms/°` | ใช้เมื่อปิด IMU — ปรับเวลาหมุนต่อองศา |
| `reset heading` | รีเซ็ตทิศทางเป็น 0 |
